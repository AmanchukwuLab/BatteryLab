# Overview
When initially repairing the system, the display and fan on R2 (liquid handling) weren't working. Upon closer inspection, some of the pins in the double row on the upper board were bent and touching each other. 

# Fix
1. Unplug the main power for the Raspberry Pi to avoid electrical hazards. 
    - Ensure all lights have turned off before proceding. 
2. Gently bend the pins to ensure none are in contact (tweezers may help).
