# Overview
The mecademic robots can be controlled manually through their web interface. This is useful for getting a sense of how they move, how they are controlled, and for determining the values of preset positions. 

Connect to a robot by entering its IP address into your web browser.
-  These IP addresses are listed inside each robot's setup file, stored in BatteryLab/BatteryLab/robots. 

At the time of this writing (Jan 15), these addresses are:
- Rail Meca500:    192.168.0.101
- Crimper Meca500: 192.168.0.100

Useful note: the position of the linear rail can be manually adjusted using the small knob on the backside of the cable inlets for the rail. Once you've set the position, CLICK the knob to ensure the rail is stationary (it can move imperceptibly slow).
